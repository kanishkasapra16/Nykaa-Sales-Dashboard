# 💄 Nykaa Sales Performance Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard built using Nykaa sales data. It provides insights into sales performance, customer behaviour, and business recommendations.

---

## 📊 Dashboard Pages

### 1️⃣ Executive Dashboard

- Total Sales
- Total Profit
- Total Orders
- Average Rating
- Average Discount
- Monthly Sales Trend
- Revenue by Category
- Top 10 Brands
- Top 10 Cities

---

### 2️⃣ Detailed Sales Performance

- Brand Analysis
- Category Revenue
- Monthly Sales Trend
- Payment Method Analysis
- Top Cities
- Product Images

---

### 3️⃣ Customer Insights

- Customer Segments
- Monthly Orders
- Customer Ratings
- Top Customer Cities
- Preferred Payment Method

---

### 4️⃣ Business Insights & Recommendations

- Business KPIs
- SWOT Analysis
- Key Insights
- Recommendations
- Conclusion

---

## 🛠 Tools Used

- Power BI
- Power Query
- DAX
- Data Visualization
